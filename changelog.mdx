<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## New Features

- API integration at `https://api.example.com/v1/docs`.

## Breaking Changes

- Updated API endpoints from `/v0` to `/v1`. Migrate using the code below.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Old endpoint (v0)
  // fetch('https://api.example.com/v0/docs');

  // New endpoint (v1)
  const response = await fetch('https://api.example.com/v1/docs', {
    headers: { Authorization: `Bearer ${YOUR_API_KEY}` }
  });
  ```
  ```python
  import requests

  # Old endpoint (v0)
  # response = requests.get('https://api.example.com/v0/docs')

  # New endpoint (v1)
  response = requests.get('https://api.example.com/v1/docs', headers={
      'Authorization': f'Bearer {YOUR_API_KEY}'
  })
  ```
</CodeGroup>

</Update>

## How to Update

Follow these steps to upgrade DAB Worx in your project.

<Steps>
  <Step title="Check Compatibility" icon="check-circle">

    Review the changelog for breaking changes.

    <Callout kind="info">
      Test in a staging environment first.
    </Callout>

  </Step>
  <Step title="Run Update" icon="download">

    ```bash
    npm update dab-worx
    ```

    Or for Yarn:

````bash
    yarn upgrade dab-worx@latest
````

  </Step>
  <Step title="Verify" icon="play-circle">

    Restart your application and test key endpoints like `https://api.example.com/v1/docs`.

  </Step>
</Steps>
