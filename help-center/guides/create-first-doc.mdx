## How to Create a Doc Page

<Steps>
  <Step title="Open your project">
    Click your project name from the dashboard.
  </Step>
  <Step title="Click 'New Page'">
    Find the "New Page" button at the top or in the sidebar.
  </Step>
  <Step title="Enter title and content">
    Add a page title, then start typing in the editor.
  </Step>
  <Step title="Save">
    Page saves automatically! Preview changes anytime.
  </Step>
</Steps>
