## Overview

DAB Worx supports seamless integrations with webhooks, APIs, and popular third-party applications. You can automate workflows, sync data, and extend functionality without custom development. Start with webhooks for real-time notifications or use the REST API for deeper custom integrations.

<Callout kind="tip">
  Review your DAB Worx plan limits for webhook and API rate limits before scaling integrations.
</Callout>

## Setting Up Webhooks

Webhooks notify external services about events in DAB Worx, such as document updates or user actions. Configure them in your project settings.

<Steps>
  <Step title="Navigate to Settings" icon="settings">
    Go to your project dashboard at `https://dashboard.example.com/projects/{projectId}/settings`.
  </Step>
  <Step title="Add Webhook" icon="plus">
    Click **Webhooks** > **New Webhook**. Enter your endpoint URL, like `https://your-webhook-url.com/dabworx`.
  </Step>
  <Step title="Select Events" icon="zap">
    Choose events: `document.updated`, `user.invited`, `page.published`.
  </Step>
  <Step title="Test and Save" icon="play">
    Send a test payload and save the webhook.
  </Step>
</Steps>

### Webhook Payload Example

Expect JSON payloads with event details.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  {
    "event": "document.updated",
    "projectId": "proj_123abc",
    "documentId": "doc_456def",
    "timestamp": "2024-10-15T10:30:00Z",
    "userId": "user_789ghi"
  ```

## API Access for Custom Integrations

Access DAB Worx data programmatically via REST API at `https://api.example.com/v1`. Authenticate with `YOUR_API_KEY`.

<ParamField header="Authorization" param-type="string" required="true">
  Bearer `{YOUR_API_KEY}`.
</ParamField>

<ParamField query="projectId" param-type="string" required="true">
  Your project identifier, like `proj_123abc`.
</ParamField>

### List Documents API

Retrieve documents from your project.

<Request tabs="cURL,JavaScript" show-lines="true">
  ```bash
  curl -X GET "https://api.example.com/v1/projects/{projectId}/documents" \
    -H "Authorization: Bearer YOUR_API_KEY"
  ```
  ```javascript
  const response = await fetch('https://api.example.com/v1/projects/proj_123abc/documents', {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  });
  const documents = await response.json();
  ```
</Request>

<Response tabs="200">
  ```json
  {
    "documents": [
      {
        "id": "doc_456def",
        "title": "API Guide",
        "updatedAt": "2024-10-15T10:30:00Z"
      }
    ],
    "total": 1
  }
  ```
</Response>
