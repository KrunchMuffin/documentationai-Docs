## Prerequisites

Before starting, ensure you have:

- A modern web browser like Chrome, Firefox, or Safari
- An email address for account verification
- Internet connection for registration

<Callout kind="info">
  No credit card or payment required to get started. You can create unlimited documentation spaces on the free plan.
</Callout>

## Create Your Account

Sign up for DAB Worx in under a minute.

<Steps>
  <Step title="Visit the Dashboard" icon="monitor">
    Navigate to `https://dashboard.dabworx.com` and click **Sign Up**.
  </Step>
  <Step title="Enter Details" icon="user">
    Provide your email, create a password, and complete the CAPTCHA.
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your inbox and click the verification link.
  </Step>
  <Step title="Log In" icon="log-in">
    Return to the dashboard and sign in with your credentials.
  </Step>
</Steps>

## Create Your First Documentation Space

Once logged in, build your project documentation quickly.

<Steps>
  <Step title="New Space" icon="plus">
    Click **New Documentation Space** in the top-right corner.
  </Step>
  <Step title="Name and Configure" icon="settings">
    Enter a name like "My Project Docs", select a template (e.g., "API Reference"), and choose your brand color `#3B82F6`.
  </Step>
  <Step title="Customize Navigation" icon="menu">
    Add pages like "Introduction" and "Quickstart" via the sidebar.
  </Step>
  <Step title="Publish" icon="globe">
    Hit **Publish** to make your space live at a URL like `https://myproject.dabworx.com`.
  </Step>
</Steps>

## Basic Interface Tour

Explore the main areas of the DAB Worx editor.

<Tabs>
  <Tab title="Sidebar" icon="sidebar">
    Manage pages, navigation, and settings.

    - **Pages**: Create, edit, delete pages
    - **Themes**: Customize colors and fonts
    - **Integrations**: Connect GitHub or webhooks
  </Tab>
  <Tab title="Editor" icon="edit-3">
    Write MDX content with live preview.

    ```mdx
    ## Welcome

    This is your first page.

    <Callout kind="tip">
      Use components for rich docs.
    </Callout>
    ```
  </Tab>
  <Tab title="Preview" icon="eye">
    Real-time rendering of your documentation.

    Toggle between desktop and mobile views to ensure responsiveness.
  </Tab>
</Tabs>

## Add Your First Content Example

Test the editor with a multi-language code snippet.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Fetch your docs API
  const response = await fetch('https://api.dabworx.com/spaces/myproject', {
    headers: {
      Authorization: `Bearer ${YOUR_API_KEY}`
    }
  });
  const data = await response.json();
  console.log(data);
  ```
  ```python
  # Fetch your docs API
  import requests

  response = requests.get(
      'https://api.dabworx.com/spaces/myproject',
      headers={'Authorization': f'Bearer {YOUR_API_KEY}'}
  )
  data = response.json()
  print(data)
  ```
</CodeGroup>
